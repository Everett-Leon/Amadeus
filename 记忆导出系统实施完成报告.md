### 1. 核心模块

**文件**：`public/memory-export-system.js`  
**大小**：~20 KB（600+ 行）  
**架构**：IIFE 模块化设计

**功能模块**：
```javascript
MemoryExportSystem
├── exportToFile()           // 导出数据到文件
├── importFromFile()         // 从文件导入数据
├── restoreData()            // 恢复数据到 LocalStorage
├── generateExportSummary()  // 生成导出摘要
├── generateImportSummary()  // 生成导入预览
└── validateImportData()     // 验证导入数据完整性
```

**导出内容**：
- ✅ 配置数据（用户配置、API 设置）
- ✅ 记忆系统（对话摘要、关键记忆、错误追踪）
- ✅ 好感度系统（好感等级、关系阶段、里程碑）
- ✅ 情感深度系统（情感等级、互动次数、脆弱时刻）
- ✅ AI 原则系统（关系健康度、行为历史、冲突记录）
- ✅ 人格一致性系统（动态特质、成长时间线、自我认知、关系数据）
- ✅ 防沉迷系统（使用统计、健康度、本周数据）

**文件**：`public/index.html`  
**位置**：设置面板 → "💾 数据管理" 区域

**UI 组件**：

### 3. 事件处理

**文件**：`public/app.js`  
**位置**：`bindSettings()` 函数

**事件绑定**：
```javascript
// 导出按钮点击
$('#btn-export-memory').addEventListener('click', handleExport);

// 导入按钮点击
$('#btn-import-memory').addEventListener('click', () => {
  $('#import-file-input').click();
});

// 文件选择
$('#import-file-input').addEventListener('change', (e) => {
  handleImport(e.target.files[0]);
});
```

**核心函数**：
- `handleExport()`：导出数据并显示摘要
- `handleImport(file)`：导入数据并恢复状态

## 📋 导出流程

```
1. 用户点击"导出所有数据"
   ↓
2. 收集所有系统数据
   - 从 LocalStorage 读取 7 个系统的数据
   - 从 config 读取用户配置
   - 生成元数据（版本、日期、完整性哈希）
   ↓
3. 处理敏感信息
   - 根据用户选择处理 API Key
   ↓
4. 生成 JSON 文件
   - 创建 Blob 对象
   - 文件名：ai-companion-memory-{日期}.json
   ↓
5. 触发下载
   - 创建临时下载链接
   - 自动点击下载
   - 清理临时资源
   ↓
6. 显示导出摘要
   - 文件名、文件大小
   - 导出内容统计
   - 成功提示
```

---

## 📥 导入流程

```
1. 用户点击"选择文件导入"
   ↓
2. 选择 JSON 文件
   - 文件大小检查（< 10MB）
   - 读取文件内容
   ↓
3. 数据验证
   - 检查 JSON 格式
   - 验证必需字段
   - 检查版本兼容性
   - 验证数据完整性
   ↓
4. 显示导入预览
   - 文件信息（文件名、导出日期、角色名）
   - 将恢复的数据列表
   - 警告信息
   - 确认提示
   ↓
5. 用户确认
   ↓
6. 备份当前数据
   - 自动备份到 LocalStorage
   - 备份 key：ai-companion-backup-{timestamp}
   ↓
7. 恢复数据
   - 写入所有系统数据到 LocalStorage
   - 更新 config
   ↓
8. 刷新 UI
   - 重新加载配置
   - 更新设置面板显示
   - 刷新所有统计数据
   ↓
9. 显示成功消息
   - 恢复的数据列表
   - 建议刷新页面
   - 询问是否立即刷新
```

### 文件安全

**建议**：
- ✅ 导出后保存到安全位置
- ✅ 不要分享给他人（包含私密对话）
- ✅ 定期更新备份

**导入警告**：
- ⚠️ 导入会覆盖当前所有数据
- ✅ 系统自动备份当前数据（可恢复）
