🌸 AI Companion 启动指南

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 最简单的启动方法：

  1. 双击 "一键启动.command"
  2. 等待终端窗口显示 "✅ 服务器已启动！"
  3. 浏览器会自动打开 http://localhost:3000
  4. 开始使用！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📂 可用的启动脚本：

  ✅ 一键启动.command           ← 推荐使用（最简单）
  
  其他脚本：
  • 启动服务器.command          （完整检查版）
  • start.command               （基础版）
  • start-background.command    （后台运行版）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🛑 如何停止服务器：

  方法 1: 关闭终端窗口
  方法 2: 在终端按 Ctrl+C
  方法 3: 双击 "stop.command"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 创建桌面快捷方式：

  1. 右键点击 "一键启动.command"
  2. 选择 "制作替身"
  3. 把替身拖到桌面
  4. 以后双击桌面图标即可启动

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
