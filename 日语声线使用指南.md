## 二、使用方法

### 1. 在欢迎页选择声线

1. 打开 http://localhost:3000
2. 展开"🎙️ 语音声线"
3. 选择"七海 (Nanami) 🌸"
4. 点击"试听"按钮，听一下效果
5. 如果满意，点击"开始通话"

### 2. 声线特点

**七海 (Nanami)** 的特点：
- **音调**：+5Hz，比普通声线略高，更清冷
- **语速**：-5%，比普通声线略慢，更有距离感
- **适合场景**：
  - 高傲、毒舌的角色（如三鹰朝）
  - 清冷、知性的角色
  - 有距离感但内心温柔的角色

### 3. 中日文混合

**重要**：
- AI 回复的是**中文**
- 但语音用的是**日语声线**
- 日语声线会用日语发音朗读中文文本
- 效果：有日语口音的中文，类似"日本人说中文"

**示例**：
- AI 回复："你好呀，今天过得怎么样？"
- 日语声线朗读：用日语发音读中文字（有日语口音）

## 三、调整参数（进阶）

如果你想进一步调整声线，可以修改 `app.js` 中的参数：

### 当前参数（七海）
```javascript
{ 
  id: 'ja-JP-NanamiNeural', 
  name: '七海 (Nanami)', 
  desc: '🌸 清冷高傲，适合三鹰朝', 
  lang: 'ja-JP', 
  pitch: '+5Hz',  // 音调
  rate: '-5%'     // 语速
}
```

### 可调整的参数

**音调 (pitch)**：
- `-10Hz` ~ `+10Hz`
- 负数：更低沉
- 正数：更清冷
- 三鹰朝建议：`+5Hz` ~ `+8Hz`

**语速 (rate)**：
- `-50%` ~ `+100%`
- 负数：更慢，更有距离感
- 正数：更快，更活泼
- 三鹰朝建议：`-5%` ~ `0%`

### 修改方法

1. 打开 `public/app.js`
2. 找到 `TTS_VOICES` 数组
3. 修改 `ja-JP-NanamiNeural` 的 `pitch` 和 `rate`
4. 保存文件，刷新浏览器

**示例**：
```javascript
// 更清冷、更慢
{ id: 'ja-JP-NanamiNeural', pitch: '+8Hz', rate: '-10%' }

// 更自然、正常语速
{ id: 'ja-JP-NanamiNeural', pitch: '+3Hz', rate: '+0%' }
```

## 五、常见问题

### Q1: 日语声线读中文听起来怪怪的？

**原因**：日语声线用日语发音读中文，会有日语口音

**解决方案**：
1. 如果不喜欢，可以用中文声线
2. 或者让 AI 回复日语（需要修改 System Prompt）

### Q2: 如何让 AI 回复日语？

修改 System Prompt，在角色设定中添加：

```
你是一个日本女生，说话时用日语。
你的母语是日语，中文不太流利。
```

然后选择日语声线，效果会更自然。

### Q3: 声线不够像三鹰朝？

可以调整参数：
- 增加音调：`pitch: '+8Hz'` 或 `'+10Hz'`
- 减慢语速：`rate: '-10%'` 或 `'-15%'`
- 或者等我们训练 GPT-SoVITS（完全克隆三鹰朝的声音）

### Q4: 试听按钮没反应？

1. 检查浏览器是否允许自动播放音频
2. 刷新页面重试
3. 查看 F12 Console 是否有错误
