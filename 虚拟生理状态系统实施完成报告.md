## ⚡ 核心机制

### 1. 状态消耗

**自然消耗（每小时）**：
- 精力：-5
- 疲劳：+3
- 心情：-2（无互动时）

**对话消耗**：
```javascript
普通对话: { energy: -2, socialBattery: -4, emotionalCapacity: -8 }
深度对话: { energy: -3, socialBattery: -5, emotionalCapacity: -10 }
冲突话题: { energy: -5, stress: +15, emotionalCapacity: -20, mood: -10 }
脆弱分享: { energy: -5, emotionalCapacity: -15, socialBattery: -10 }
愉快互动: { energy: -1, mood: +5, socialBattery: -3 }
```

**清醒过久惩罚**：
- 16小时+：energy -10/小时，fatigue +15/小时
- 24小时+：energy -20，fatigue +30，stress +15

### 2. 状态恢复

**休息（每小时）**：
- 精力：+15
- 疲劳：-10
- 压力：-8
- 社交电量：+20
- 情绪容量：+10

**睡眠（完整周期）**：
- 精力：100（完全恢复）
- 疲劳：0
- 压力：-50
- 社交电量：100
- 情绪容量：100
- 清醒时长：重置为 0

**独处充电（30分钟）**：
- 社交电量：+30
- 压力：-5

### 3. 响应调整

**精力值影响响应长度**：
- 70-100：完整响应（100%）
- 40-69：适度响应（70%）
- 20-39：简短响应（40%）
- 0-19：极简响应（20%）

**状态注入 System Prompt**：
```
【当前生理状态】
你现在精力充沛，状态很好。

根据你的生理状态调整回复：
- 精力充沛时：回复详细、情感丰富、主动分享
- 疲惫时：回复简短、情感平淡、少主动
- 非常疲惫时：自然地表达需要休息
- 压力大时：情绪紧张、耐心降低
- 社交电量低时：倾向独处、委婉表达需要充电
```
