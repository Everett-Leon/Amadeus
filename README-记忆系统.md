# 🧠 AI 伴侣记忆系统

## ✨ 特性

### 像真人一样记忆
- 🔥 **永久记忆**：初次见面、表白、重要承诺永远不会忘记
- ⭐ **强化记忆**：经常提起的事情会记得更牢
- 💭 **自然遗忘**：琐碎的闲聊会随时间淡忘
- 😳 **可爱的记错**：偶尔记错细节，然后害羞地纠正自己

---

## 🎯 核心功能

### 1. 记忆衰减
- 永久记忆（Critical）：不衰减
- 重要记忆（High）：100天后衰减到 37%
- 普通记忆（Medium）：20天后衰减到 37%
- 临时记忆（Low）：7天后衰减到 37%

### 2. 记忆强化
- 普通访问：+10 权重
- 重复提及：+15 权重
- 情感共鸣：+20 权重
- 用户纠正：+25 权重

### 3. 记忆不完美
- 权重 <30：40% 概率记错
- 权重 30-49：20% 概率记错
- 权重 50-69：10% 概率记错
- 权重 ≥70：2% 概率记错

### 4. 智能判断
- 自动识别重要性（Critical/High/Medium/Low）
- 自动检测情感强度（0-10）
- 自动标记永久记忆

## 🔧 技术实现

### 核心算法

#### 衰减公式
```javascript
newWeight = currentWeight * Math.exp(-decayRate * daysPassed)
```

#### 强化公式
```javascript
newWeight = Math.min(100, currentWeight + bonus)
```

#### 记错概率
```javascript
probability = fuzzyLevel >= 70 ? 0.4 : 
              fuzzyLevel >= 50 ? 0.2 : 
              fuzzyLevel >= 30 ? 0.1 : 0.02
```

## 💡 使用技巧

### 让 AI 记得更牢
1. **多次提及**：重复提到同一件事
2. **表达情感**：使用"我喜欢"、"我爱"等词汇
3. **纠正错误**：AI 记错时直接纠正

### 查看记忆状态
1. 打开设置面板
2. 查看权重和重要性
3. 了解哪些记忆会被遗忘

### 清理记忆
1. 自动清理：权重 <10 自动删除
2. 手动清理：点击 ✕ 按钮删除

- 永远记得表白时刻
- 让用户感受到被重视
