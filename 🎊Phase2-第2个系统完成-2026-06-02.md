## ⚡ 核心特性

### 3 种打断检测

| 方式 | 触发时机 | 检测延迟 | 效果 |
|------|----------|----------|------|
| **语音打断** | 用户开始说话 | < 50ms | AI 立即停止 |
| **打字打断** | 用户开始输入 | < 50ms | AI 立即停止 |
| **点击麦克风** | 用户点击麦克风按钮 | < 50ms | AI 立即停止 |

### AI 打断策略

**紧急关键词**：
- 危险、小心、注意、等等、别
- danger, careful, wait, stop

**情绪触发**：
- angry（生气）
- surprised（惊讶）

**限制**：
- 每小时最多打断 3 次
- 仅紧急情况打断

### 打断记录

**用户打断**：
```javascript
{
  type: 'user',
  source: 'speech',          // 'speech' | 'typing' | 'mic'
  timestamp: 1717339200000,
  aiText: 'AI 正在说...',
  aiDuration: 1500,          // AI 已说话时长（ms）
}
```

**AI 打断**：
```javascript
{
  type: 'ai',
  reason: 'urgent',
  timestamp: 1717339200000,
}
```
