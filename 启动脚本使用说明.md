# 🚀 启动脚本使用说明

## 📋 可用脚本

| 脚本文件 | 用途 | 说明 |
|---------|------|------|
| **start.command** | 标准启动 | 双击启动，窗口关闭时服务器停止 ✅ 推荐 |
| **start-background.command** | 后台启动 | 后台运行，关闭窗口后继续运行 |
| **stop.command** | 停止服务器 | 停止所有运行中的服务器 |
| **quick-start.sh** | 快速启动 | 命令行使用，适合开发者 |

---

## 1️⃣ 标准启动（推荐）⭐

### 使用方法
1. **双击** `start.command` 文件
2. 终端窗口会自动打开
3. 浏览器自动打开 http://localhost:3000
4. 关闭终端窗口即可停止服务器

### 特点
- ✅ 自动检查 Node.js 安装
- ✅ 自动安装依赖（首次运行）
- ✅ 自动检测端口冲突
- ✅ 自动打开浏览器
- ✅ 实时查看服务器日志
- ✅ 关闭窗口自动停止服务器

### 首次使用提示
如果双击后提示"无法打开"，请：
```bash
# 在终端中执行：
cd /Users/luofanchen/Desktop/Amadeus/ai-companion
chmod +x start.command
```

---

## 2️⃣ 后台启动

### 使用方法
1. **双击** `start-background.command` 文件
2. 服务器在后台启动
3. 可以关闭终端窗口
4. 服务器继续运行

### 停止服务器
- **方法 1**：双击 `stop.command`
- **方法 2**：在终端运行 `kill $(cat logs/server.pid)`

### 查看日志
```bash
tail -f logs/server.log
```

---

## 3️⃣ 停止服务器

### 使用方法
1. **双击** `stop.command` 文件
2. 自动停止所有运行中的服务器

### 手动停止（高级）
```bash
# 查找进程
lsof -ti:3000

# 停止进程
lsof -ti:3000 | xargs kill -9
```

## 💡 使用技巧

### 技巧 1：创建桌面快捷方式
1. 右键 `start.command`
2. 选择"制作替身"
3. 将替身拖到桌面
4. 以后双击桌面图标即可启动

### 技巧 2：开机自动启动
1. 打开"系统偏好设置" → "用户与群组"
2. 选择"登录项"
3. 点击"+" 添加 `start-background.command`

### 技巧 3：查看服务器状态
```bash
# 检查服务器是否运行
lsof -i:3000

# 查看进程详情
ps aux | grep "node server.js"
```

### 技巧 4：更改端口
如果需要更改端口（默认 3000）：
1. 编辑 `server.js`
2. 修改 `const PORT = 3000;` 为其他端口
3. 同步修改启动脚本中的端口号
