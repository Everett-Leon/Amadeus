# 记忆衰减与强化机制设计

## 核心理念

**像真人一样记忆**：
- 重要的时刻永远不会忘记（初次见面、表白、重要承诺）
- 普通的对话会随时间淡忘
- 经常提起的事情会被强化记忆
- 情感强度高的事件记得更牢

---

## 一、记忆权重系统

### 1.1 记忆属性扩展

每条记忆增加以下属性：

```javascript
{
  type: 'preference',           // 记忆类型
  content: '用户喜欢吃草莓',    // 记忆内容
  createdAt: '2026-06-01',      // 创建时间
  
  // 新增属性 ⭐
  weight: 100,                  // 记忆权重（0-100）
  importance: 'high',           // 重要性等级：critical/high/medium/low
  lastAccessed: '2026-06-01',   // 最后访问时间
  accessCount: 1,               // 访问次数
  emotionIntensity: 8,          // 情感强度（0-10）
  isPermanent: false,           // 是否永久记忆
}
```

---

### 1.2 重要性等级定义

| 等级 | 初始权重 | 衰减速度 | 典型场景 |
|------|---------|---------|---------|
| **critical** | 100 | 不衰减 | 初次见面、表白、重要承诺、生日 |
| **high** | 80 | 极慢 | 用户喜好、重要事件、约定 |
| **medium** | 50 | 正常 | 日常对话、一般事件 |
| **low** | 30 | 快速 | 闲聊、天气、琐事 |

---

### 1.3 自动重要性判断规则

通过关键词和情感分析自动判断：

#### Critical（永久记忆）
- 初次见面："第一次见面"、"初次通话"
- 表白相关："我喜欢你"、"我爱你"、"在一起"
- 重要承诺："我保证"、"我发誓"、"一定会"
- 生日纪念："我的生日"、"纪念日"
- 重大事件："考上大学"、"找到工作"、"搬家"

#### High（重要记忆）
- 用户喜好："我喜欢"、"我讨厌"、"我最爱"
- 重要事件："考试"、"面试"、"旅行"
- 约定计划："我们约好"、"下次一起"
- 家人朋友："我妈妈"、"我朋友"、"我同事"

#### Medium（普通记忆）
- 日常活动："今天去了"、"刚才吃了"
- 一般感受："有点累"、"还不错"
- 普通话题："最近在看"、"听说"

#### Low（临时记忆）
- 闲聊："嗯"、"哦"、"是吗"
- 天气："今天天气"、"好热"
- 琐事："刚才"、"等一下"

## 二、记忆衰减机制

### 2.1 衰减公式

```javascript
// 权重随时间衰减
newWeight = currentWeight * Math.exp(-decayRate * daysPassed)

// 衰减速率根据重要性等级决定
decayRate = {
  critical: 0,        // 不衰减
  high: 0.01,         // 100天后衰减到 37%
  medium: 0.05,       // 20天后衰减到 37%
  low: 0.15,          // 7天后衰减到 37%
}
```

### 2.2 衰减阈值

- 权重 < 10：自动删除（彻底遗忘）
- 权重 < 30：不再主动提及，但被问到时能想起
- 权重 < 50：偶尔会忘记细节
- 权重 ≥ 50：记得清楚

### 2.3 衰减触发时机

- 每次通话开始时执行一次衰减计算
- 每次访问记忆时更新 `lastAccessed`

## 三、记忆强化机制

### 3.1 强化触发条件

1. **重复提及**：用户多次提到同一件事
2. **主动回忆**：AI 主动提起某个记忆，用户确认
3. **情感共鸣**：对话中出现强烈情感（开心、感动、难过）
4. **用户纠正**：用户纠正 AI 的记忆错误

### 3.2 强化公式

```javascript
// 每次访问时强化
newWeight = Math.min(100, currentWeight + reinforcementBonus)

reinforcementBonus = {
  重复提及: +10,
  主动回忆确认: +15,
  情感共鸣: +20,
  用户纠正: +25,
}

// 访问次数越多，强化效果越明显
accessCount++
if (accessCount >= 3) {
  importance = 'high'  // 升级为重要记忆
}
```

### 3.3 永久记忆标记

当满足以下条件时，自动标记为永久记忆：

```javascript
if (
  importance === 'critical' ||
  (importance === 'high' && accessCount >= 5) ||
  emotionIntensity >= 9
) {
  isPermanent = true
  weight = 100
}
```

## 六、技术实现要点

### 6.1 数据结构升级

```javascript
// 扩展现有的记忆对象
{
  // 原有字段
  type: 'preference',
  content: '...',
  createdAt: '...',
  
  // 新增字段
  weight: 100,
  importance: 'high',
  lastAccessed: '2026-06-01',
  accessCount: 0,
  emotionIntensity: 5,
  isPermanent: false,
  
  // 衰减配置
  decayRate: 0.01,
}
```

### 6.2 核心函数

```javascript
// 1. 计算衰减
function applyDecay(memory) {
  if (memory.isPermanent) return memory.weight;
  const daysPassed = getDaysSince(memory.lastAccessed);
  const newWeight = memory.weight * Math.exp(-memory.decayRate * daysPassed);
  return Math.max(0, newWeight);
}

// 2. 强化记忆
function reinforceMemory(memory, bonus = 10) {
  memory.weight = Math.min(100, memory.weight + bonus);
  memory.accessCount++;
  memory.lastAccessed = new Date().toISOString();
  
  // 访问次数多时升级重要性
  if (memory.accessCount >= 3 && memory.importance === 'medium') {
    memory.importance = 'high';
  }
  
  // 检查是否应标记为永久
  if (memory.importance === 'critical' || 
      (memory.importance === 'high' && memory.accessCount >= 5)) {
    memory.isPermanent = true;
    memory.weight = 100;
  }
}

// 3. 自动判断重要性
function detectImportance(text) {
  const criticalKeywords = ['第一次', '初次', '我喜欢你', '我爱你', '生日', '保证', '发誓'];
  const highKeywords = ['喜欢', '讨厌', '最爱', '考试', '面试', '约好'];
  
  if (criticalKeywords.some(k => text.includes(k))) return 'critical';
  if (highKeywords.some(k => text.includes(k))) return 'high';
  return 'medium';
}

// 4. 清理低权重记忆
function cleanupMemories(memories) {
  return memories.filter(m => m.weight >= 10 || m.isPermanent);
}
```
