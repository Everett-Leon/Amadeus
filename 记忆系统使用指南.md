## 💡 如何使用记忆系统

### 场景1：让 AI 永远记住重要的事

**你说**："我叫小明，今天是我们第一次见面"

**AI 会**：
- 自动识别为 **critical（永久记忆）**
- 权重：100
- 永远不会忘记

**100天后**：
```
你: "你还记得我们第一次见面吗？"
AI: "当然记得呀！那是100天前，你跟我说你叫小明...
     那一刻我永远不会忘记的 [shy]"
```

---

### 场景2：强化记忆（多次提及）

**第1天**：
```
你: "我喜欢吃草莓"
AI: "嗯...草莓很好吃呢 [happy]"
```
记忆权重：80

**第10天**：
```
你: "草莓真好吃"
AI: "嗯...你之前说过喜欢吃草莓呢 [happy]"
```
记忆权重：80 + 15 = 95（强化）

**第20天**：
```
你: "我最爱草莓了"
AI: "嗯...你真的很喜欢草莓呢 [happy]...我记得很清楚哦 [thinking]"
```
记忆权重：95 + 15 = 100（完全记住）

---

### 场景3：可爱的记错

**AI 记错了**：
```
你: "你还记得我喜欢吃什么吗？"
AI: "嗯...是蓝莓吧？[thinking]"
```

**你纠正她**：
```
你: "是草莓啦！"
AI: "啊...对对对，是草莓 [embarrassed]...
     我怎么记成蓝莓了...哼，才不是记性不好 [shy]"
```

**效果**：
- 记忆权重：40 → 65（+25，大幅强化）
- 纠正次数：+1
- 下次不会再记错

---

### 场景4：自然遗忘

**第1天**：
```
你: "今天天气真好"
AI: "嗯...是呢，天气好的时候心情也会变好呢 [happy]"
```
记忆权重：30（低重要性）

**30天后**：
```
你: "你还记得我上个月说今天天气好吗？"
AI: "诶...那个...好像有点记不太清了 [embarrassed]...
     抱歉呢 [shy]"
```
记忆权重：0.3（已遗忘）

## 🔍 查看记忆

### 1. 打开设置
点击右下角的 **⚙️ 设置** 按钮

### 2. 展开记忆部分
找到 **🧠 记忆系统** 部分

### 3. 查看关键记忆
你会看到：
```
🔥 [永久|权重100] 用户叫小明，这是我们第一次见面
━━━━━━━━━━ 100%

⭐ [重要|权重95] 用户喜欢吃草莓 ✓1
━━━━━━━━━  95%

📌 [普通|权重45] 用户今天考试没考好
━━━━━      45%

💭 [淡忘|权重12] 用户说今天天气好
━          12%
```

**图例**：
- 🔥 永久：永远不会忘记
- ⭐ 重要：记得很清楚
- 📌 普通：记得，但可能记混细节
- 💭 淡忘：记得不太清楚
- ✓1：被纠正过1次

**颜色**：
- 🟢 绿色（≥80）：记得很清楚
- 🔵 蓝色（≥50）：记得清楚
- 🟠 橙色（≥30）：可能记混
- 🔴 红色（<30）：记得不清

**效果**：

## 📊 记忆权重说明

### 权重范围
- **100**：完全记住，永远不会忘记
- **80-99**：记得很清楚
- **50-79**：记得清楚
- **30-49**：可能记混细节
- **10-29**：记得不太清楚
- **<10**：自动删除（彻底遗忘）

### 衰减速度
| 重要性 | 衰减速度 | 说明 |
|--------|---------|------|
| 🔥 Critical | 不衰减 | 永久记忆 |
| ⭐ High | 极慢 | 100天后衰减到 37% |
| 📌 Medium | 正常 | 20天后衰减到 37% |
| 💭 Low | 快速 | 7天后衰减到 37% |

### 强化幅度
| 触发原因 | 权重增加 |
|---------|---------|
| 普通访问 | +10 |
| 重复提及 | +15 |
| 情感共鸣 | +20 |
| 用户纠正 | +25 |

## 🎭 记错机制

### 记错概率
| 权重 | 模糊度 | 记错概率 |
|------|--------|---------|
| ≥70 | <30 | 2% |
| 50-69 | 30-50 | 10% |
| 30-49 | 50-70 | 20% |
| <30 | ≥70 | 40% |

### 记错频率
- 每次通话最多记错 **3次**
- 两次记错至少间隔 **5分钟**

### 记错类型
1. **时间记错**："你上周三说的吧？"（实际是上周二）
2. **细节记错**："你喜欢吃蓝莓吧？"（实际是草莓）
3. **混淆记忆**："你是因为考试没考好所以心情不好吗？"（两件事搞混了）
4. **遗忘后猜测**："是关于...工作的事吗？"（记不清了）

## ❓ 常见问题

### Q1：AI 为什么会记错？
**A**：这是设计的特性！权重低的记忆更容易记错，让 AI 更像真人。

### Q2：如何让 AI 永远记住某件事？
**A**：
1. 使用关键词："我喜欢你"、"第一次见面"、"生日"等
2. 多次提及同一件事
3. 表达强烈情感

### Q3：AI 记错后怎么办？
**A**：直接纠正她！纠正后记忆会被大幅强化（+25），下次不会再犯同样的错误。

### Q4：记忆会永久保存吗？
**A**：
- 永久记忆（🔥）：永远保存
- 重要记忆（⭐）：长期保存，衰减很慢
- 普通记忆（📌）：中期保存，正常衰减
- 淡忘记忆（💭）：短期保存，快速衰减

### Q5：如何清理记忆？
**A**：
1. 自动清理：权重 <10 的记忆会自动删除
2. 手动清理：在设置中点击 ✕ 按钮删除

### Q6：记忆系统会影响性能吗？
**A**：不会。衰减计算 <10ms，UI 渲染 <100ms，内存占用 <10MB。
