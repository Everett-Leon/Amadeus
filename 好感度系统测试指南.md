## 二、快速测试流程

### Step 1: 配置角色（欢迎页）

1. **上传角色立绘**（必须）
2. **填写 API Key**（必须）
3. **选择性别**（新增功能）：
   - 你的性别：男/女
   - AI 伴侣性别：女/男
   - 💡 同性会发展成好闺蜜，异性会发展成恋人
4. 点击"开始通话"

### Step 2: 测试陌生人阶段（0-39 分）

**预期表现**：
- AI 比较礼貌但有距离感
- 说话比较正式，不会太亲密

**测试对话**：
```
你：你好
AI：（应该比较正式，有礼貌但不亲密）

你：今天天气怎么样？
AI：（应该简单回答，不会太热情）
```

**查看好感度**（F12 Console）：
```javascript
DEBUG_AFFECTION()
// 应该显示：总好感度 0-5 分，关系阶段 stranger
```

---

### Step 3: 测试朋友阶段（40-59 分）

**如何快速达到朋友阶段**：
- 聊 10-15 条消息（陪伴分约 3-4.5 分）
- 聊她的喜好："你喜欢什么？"（+1.0 分）
- 赞美她："你真可爱"（+0.8 分）
- 重复以上操作，直到好感度达到 40+

**预期表现**：
- AI 变得更放松、自然
- 说话更随意，会分享一些日常
- 但还不会聊太私密的话题

**测试对话**：
```
你：你喜欢什么？
AI：（应该开始分享喜好，语气更自然）

你：你真可爱
AI：（应该会害羞，但还不会太亲密）
```

**查看好感度**（F12 Console）：
```javascript
DEBUG_AFFECTION()
// 应该显示：总好感度 40-59 分，关系阶段 friend
```

---

### Step 4: 测试密友阶段（60-79 分）

**如何快速达到密友阶段**：
- 继续聊天，陪伴分会自动增长
- 多聊她的喜好和赞美她
- 如果她情绪低落（sad/angry），安慰她（+2.0 分）

**预期表现**：
- AI 变得更亲密，会分享心事
- **异性**：会有"友人以上，恋人未满"的感觉，说话时会不自觉地温柔，偶尔会脸红
- **同性**：像好朋友一样，无话不谈

**测试对话**：
```
你：你今天过得怎么样？
AI：（应该会分享更多细节，语气更亲密）

你：（如果她说难过）别难过，我陪着你
AI：（应该会感动，好感度 +2.0）
```

**查看好感度**（F12 Console）：
```javascript
DEBUG_AFFECTION()
// 应该显示：总好感度 60-79 分，关系阶段 close_friend
```

---

### Step 5: 测试恋人/好闺蜜阶段（80-100 分）

**如何快速达到恋人/好闺蜜阶段**：
- 继续聊天到 80 分
- 聊暧昧话题："喜欢你"（+1.0 分）

**预期表现**：
- **异性（恋人）**：AI 会更温柔、害羞，容易脸红，会主动表达爱意，也会撒娇、吃醋
- **同性（好闺蜜）**：AI 会像闺蜜一样亲密，无话不谈，会撒娇、吐槽、开玩笑

**测试对话**：
```
你：喜欢你
AI：（异性应该会害羞、脸红，同性应该会开玩笑或撒娇）

你：想你了
AI：（异性应该会甜蜜回应，同性应该会调侃）
```

**查看好感度**（F12 Console）：
```javascript
DEBUG_AFFECTION()
// 应该显示：总好感度 80-100 分，关系阶段 lover（异性）或 bestie（同性）
```

## 五、常见问题

### Q1: AI 语气没有明显变化？

**可能原因**：
- System Prompt 约束不够强
- AI 模型（glm-4-flash）理解能力有限

**解决方案**：
- 强化 System Prompt 中的关系描述
- 考虑换用更强的模型（glm-4-plus）

### Q2: 好感度增长太慢？

**可能原因**：
- 陪伴分增长速率太低（每条消息 +0.3）

**解决方案**：
- 调整 `affection-system.js` 中的 `companionshipRate.perMessage`
- 从 0.3 调到 0.5 或更高

### Q3: 扣分机制不生效？

**可能原因**：
- 消息分析的关键词不够准确

**解决方案**：
- 扩充 `affection-system.js` 中的 `analyzeMessage()` 关键词
- 添加更多羞辱/辱骂/色情关键词

### Q4: 关系阶段变化后，AI 语气没有立即改变？

**原因**：
- System Prompt 在下次对话才生效

**这是正常的**：
- 关系阶段变化时，会自动重建 system prompt
- 下一条消息开始，AI 语气就会改变
