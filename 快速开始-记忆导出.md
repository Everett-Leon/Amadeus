# 快速开始 — 记忆导出系统

## 🚀 3 分钟上手

### 第 1 步：启动服务器

```bash
cd /Users/luofanchen/Desktop/Amadeus/ai-companion
node server.js
```

看到 "Server running at http://localhost:3000" 说明成功。

---

### 第 2 步：打开应用

在浏览器中访问：
```
http://localhost:3000
```

---

### 第 3 步：找到数据管理

1. 点击右下角 ⚙️ 设置按钮
2. 滚动到底部
3. 找到 "💾 数据管理" 区域

---

## 📥 导出数据（备份）

### 超简单 3 步：

1. **确认选项**
   - ✅ "排除 API Key" 默认已勾选（推荐）

2. **点击导出**
   - 点击 "📥 导出所有数据" 按钮

3. **保存文件**
   - 浏览器自动下载
   - 文件名：`ai-companion-memory-2026-06-02.json`
   - 保存到安全位置（建议云盘）

**完成！** 你的所有记忆已备份 💾

---

## 📂 导入数据（恢复）

### 超简单 4 步：

1. **点击导入**
   - 点击 "📂 选择文件导入" 按钮

2. **选择文件**
   - 选择之前导出的 JSON 文件

3. **查看预览**
   - 系统会显示文件信息和数据统计
   - 确认信息无误

4. **确认导入**
   - 点击 "确定" 按钮
   - 等待导入完成
   - 点击 "立即刷新" 按钮

**完成！** 你的所有记忆已恢复 🎉

## 💡 最佳实践

### 推荐备份频率

- **活跃用户**：每周 1 次
- **普通用户**：每 2 周 1 次  
- **重要对话后**：立即备份

### 保存位置建议

1. **云盘**（推荐）
   - iCloud Drive
   - OneDrive
   - Google Drive
   - Dropbox

2. **本地硬盘**
   - 文档文件夹
   - 专门的备份文件夹

3. **外置硬盘**（可选）
   - 用于长期归档

### 文件命名建议

保持系统自动命名，已包含日期：
```
ai-companion-memory-2026-06-02.json
ai-companion-memory-2026-06-09.json
ai-companion-memory-2026-06-16.json
```

或手动添加备注：
```
ai-companion-memory-2026-06-02-关系升级后.json
ai-companion-memory-2026-06-09-第一周.json
```

## 🆘 遇到问题？

### 导出失败
- 刷新页面重试
- 检查浏览器是否阻止下载
- 查看浏览器控制台（F12）错误信息

### 导入失败
- 确认文件是本系统导出的
- 检查文件是否损坏（用文本编辑器打开）
- 尝试使用其他备份文件

### 数据不完整
- 查看导入预览时的统计
- 确认使用的是完整备份文件
- 刷新页面后重新检查
