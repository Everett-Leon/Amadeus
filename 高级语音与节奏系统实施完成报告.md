### 3. 响应调整流程

```javascript
1. 获取情绪和生理状态
   ↓
2. 计算节奏参数（延迟/速度/停顿/长度）
   ↓
3. 应用情绪修正器
   ↓
4. 应用生理状态修正器
   ↓
5. 调整响应内容（长度/前缀/后缀）
   ↓
6. 添加响应延迟（模拟思考）
   ↓
7. 显示处理后的响应
```

## 🌟 创新点

### 1. 多维度节奏控制

不是单一的"快"或"慢"，而是：
- **响应延迟** - 思考时间
- **打字速度** - 说话速度
- **思考停顿** - 句间停顿
- **回复长度** - 内容详细度

### 2. 情绪与生理状态联动

```
情绪（开心） + 生理（精力充沛） = 快速详细
情绪（难过） + 生理（疲惫） = 极慢极简
情绪（生气） + 生理（高压力） = 急促简短
```

### 3. 自然的情感表达

添加情感前缀：
- *沉默片刻* （难过时）
- *思考中...* （思考时）
- *眼睛睁大* （惊讶时）
- *眼皮很沉* （疲惫时）
- *深呼吸* （压力大时）

### 4. 智能长度调整

根据状态自动缩短或保持：
- 疲惫时：极简短（20% 原长度）
- 正常时：适中（100%）
- 开心时：详细（130%）

### 5. 渐进式体验

状态变化是渐进的：
- 精力 100 → 80 → 60 → 40 → 20 → 10
- 每个阶段节奏不同
- 用户可以明显感知变化

## 💡 使用示例

### 代码调用

```javascript
// 1. 计算节奏参数
const rhythm = AdvancedSpeechRhythmSystem.calculateRhythm({
  emotion: 'sad',
  physiologicalState: PhysiologicalStateSystem.getState(),
  messageLength: 150,
  contextComplexity: 'normal',
});

// 2. 应用节奏到响应
const processed = AdvancedSpeechRhythmSystem.applyRhythmToResponse(
  originalResponse,
  rhythm,
  { addPrefix: true, adjustLength: true }
);

// 3. 添加延迟
await sleep(rhythm.responseDelay);

// 4. 显示
display(processed);
```

### 参数输出示例

```javascript
{
  responseDelay: 2000,        // 2秒延迟
  typingSpeed: 20,            // 20字/秒
  thinkingPause: 3000,        // 3秒停顿
  maxLength: 200,             // 最多200字
  prefix: '*沉默片刻*\n',    // 前缀
  suffix: '',                 // 后缀
}
```
