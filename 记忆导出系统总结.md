## 📦 系统概览

### 核心功能

1. **📥 完整导出**
   - 一键打包所有 7 个系统的数据
   - 生成 JSON 文件，自动下载
   - 文件大小：20KB - 2MB（取决于使用时长）
   - 导出时间：< 100ms（瞬间完成）

2. **📂 智能导入**
   - 读取 JSON 文件并验证
   - 显示详细导入预览
   - 自动备份当前数据
   - 恢复所有系统状态
   - 导入时间：< 500ms

3. **🔒 隐私保护**
   - 默认排除 API Key
   - 所有操作本地完成
   - 不上传任何服务器
   - 用户完全控制数据流向

4. **🛡️ 安全保障**
   - 数据完整性验证（长度哈希）
   - 版本兼容性检查
   - 必需字段验证
   - 导入前自动备份
   - 错误恢复机制

## 📋 导出数据结构

### 元数据
```json
{
  "metadata": {
    "version": "1.0",                          // 数据格式版本
    "exportDate": "2026-06-02T16:30:00.000Z", // 导出时间
    "appVersion": "0.3",                       // 应用版本
    "characterName": "三鹰朝",                 // 角色名
    "exportType": "full",                      // 导出类型
    "dataIntegrity": "length:152847"          // 完整性标记
  }
}
```

### 包含的系统（7个）

1. **用户配置** (`config`)
   - 角色设定、API 配置、声线选择、性别设置、记忆配置

2. **记忆系统** (`memorySystem`)
   - 对话摘要、关键记忆、错误追踪

3. **好感度系统** (`affectionSystem`)
   - 好感等级、关系阶段、互动次数、里程碑

4. **情感深度系统** (`emotionalDepthSystem`)
   - 情感等级、相处天数、情感分享、脆弱时刻、互相关心

5. **AI 原则系统** (`aiPrinciplesSystem`)
   - 关系健康度、行为历史、冲突记录、和解历史

6. **人格一致性系统** (`personalityConsistencySystem`)
   - 动态特质、成长时间线、自我认知、重要时刻、内部笑话

7. **防沉迷系统** (`antiAddictionSystem`)
   - 使用统计、健康度、本周数据、偏好设置

```javascript
// 导出时生成
const dataStr = JSON.stringify(data);
data.metadata.dataIntegrity = `length:${dataStr.length}`;

// 导入时验证
const expectedLength = parseInt(data.metadata.dataIntegrity.split(':')[1]);
const actualLength = JSON.stringify(data).length;

}
```
