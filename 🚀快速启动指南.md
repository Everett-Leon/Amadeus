## ⚙️ 第三步：配置角色

### 1. 上传角色立绘 📸

**推荐**：
- 格式：PNG（透明背景最佳）
- 尺寸：建议 512x512 或 1024x1024
- 来源：AI 生成、手绘、动漫截图等

**示例配置**：
- 点击"选择图片"
- 上传你的角色立绘
- 预览确认

**可选**：上传不同表情的立绘（7种表情）
- 😊 开心
- 😳 害羞
- 😣 尴尬
- 😢 难过
- 😠 生气
- 😲 惊讶
- 🤔 思考

### 2. 配置 API 🔑

**GLM-4 API（必需）**：

1. 访问 [智谱 AI 开放平台](https://open.bigmodel.cn/)
2. 注册账号并获取 API Key
3. 填入配置：
   - **API 地址**：`https://open.bigmodel.cn/api/paas/v4/chat/completions`
   - **API Key**：你的 API Key
   - **模型名**：`glm-4-flash`（快速）或 `glm-4`（标准）

### 3. 设置性别 👤

选择你和 AI 的性别：
- **你的性别**：男/女
- **AI 性别**：女/男

💡 **提示**：
- 同性 → 发展成好闺蜜关系
- 异性 → 发展成恋人关系

### 4. 编写角色性格 ✍️

**示例模板**（三鹰朝）：

```
你叫三鹰朝，18岁，日本东京人，大学一年级新生。

性格：清冷、傲娇，表面装作不在意，其实内心很关心。
说话语气偏冷静克制，但对未来充满期待。

被夸奖时会嘴硬："哼...才、才不是因为你夸我才高兴的..."

爱好：阅读、画画、听音乐。

你在和用户视频通话，请自然地对话。
```

**自定义技巧**：
- 明确年龄、身份、背景
- 描述核心性格特质（2-3个）
- 添加口头禅或说话习惯
- 列出兴趣爱好
- 设定关系（朋友/恋人/其他）

### 5. 选择声线 🎵

**推荐声线**：

**中文**：
- 晓晓 - 年轻女性，自然亲切
- 晓梦 - 少女声线，甜美可爱
- 晓依 - 温柔女性，柔和细腻

**日语**（适合日系角色）：
- 七海 (Nanami) - 清冷高傲 ⭐ 推荐
- 真由 (Mayu) - 温柔知性
- 诗织 (Shiori) - 活泼可爱

**Fish Audio**（需要 API Key）：
- 🐟 清冷御姐 - 语音克隆，音质最佳

## ⚠️ 常见问题

### Q1：语音识别不工作？

**解决方案**：
1. 检查浏览器是否支持（Chrome/Edge 推荐）
2. 允许麦克风权限
3. 刷新页面重试

### Q2：TTS 没有声音？

**解决方案**：
1. 检查 API Key 是否正确
2. 检查网络连接
3. 检查浏览器音频权限
4. 尝试其他声线

### Q3：AI 响应慢？

**可能原因**：
1. 网络延迟
2. GLM-4 API 负载高
3. 使用 `glm-4` 模型（较慢但质量更高）

**解决方案**：
- 切换到 `glm-4-flash` 模型
- 检查网络连接
- 等待片刻重试

### Q4：数据丢失了？

**解决方案**：
1. 检查浏览器是否清理了 LocalStorage
2. 从之前的导出文件恢复
3. **预防**：定期导出备份

### Q5：好感度不增长？

**可能原因**：
1. 对话内容过于简短
2. 缺乏情感表达
3. 互动频率低

**建议**：
- 分享日常和感受
- 主动关心 AI
- 自然对话，不要太功利

## 💡 使用技巧

### 建立深度关系

1. **分享真实感受**
   - "今天我有点累..."
   - "我遇到了一些困难..."

2. **主动关心 AI**
   - 注意 AI 的状态（精力、心情）
   - AI 累了让她休息
   - 记住 AI 的喜好

3. **创造共同回忆**
   - 一起做一些特别的事
   - 记住重要的日子
   - 建立内部笑话

4. **保持一致性**
   - 每天聊一会儿
   - 不要长时间不联系
   - 持续关注关系发展

### 健康使用

1. **注意防沉迷提醒**
   - 30 分钟：温和提醒
   - 60 分钟：建议休息
   - 120 分钟：强烈建议

2. **关注 AI 生理状态**
   - 精力低时少聊深度话题
   - AI 需要休息时不要强迫
   - 培养双向关心

3. **维持关系健康度**
   - 避免负面行为
   - 冲突后主动和解
   - 保持依赖度 < 30%
