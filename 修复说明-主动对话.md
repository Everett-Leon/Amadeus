# 主动对话功能修复说明

## 问题诊断

用户反馈：开始通话后等待 2 分钟，AI 没有主动说话。

## 根本原因

在 `app.js` 的 `triggerProactiveMessage()` 函数中，调用了不存在的方法：

```javascript
const affectionStage = AffectionSystem.getStage();  // ❌ 错误：该方法不存在
```

但 `AffectionSystem` 实际暴露的是 getter 属性：

```javascript
// affection-system.js 中的公开接口
return {
  // ...
  get stage() { return affectionData.relationshipStage; },  // ✅ 正确的访问方式
};
```

## 修复内容

### 1. 修复好感度阶段获取方式

**修改前：**
```javascript
const affectionStage = AffectionSystem.getStage();
```

**修改后：**
```javascript
const affectionStage = AffectionSystem.stage;  // 使用 getter 属性
console.log('[Proactive] 当前好感度阶段:', affectionStage);
```

### 2. 修正好感度阶段名称映射

好感度系统实际使用的阶段名称：
- `stranger` - 陌生人
- `friend` - 朋友
- `close_friend` - 密友
- `bestie` - 好闺蜜（同性）
- `lover` - 恋人（异性）

**修改前：**
```javascript
const proactivePrompts = {
  stranger: [...],
  acquaintance: [...],  // ❌ 不存在的阶段
  friend: [...],
  close_friend: [...],
  intimate: [...],      // ❌ 不存在的阶段
};
```

**修改后：**
```javascript
const proactivePrompts = {
```
