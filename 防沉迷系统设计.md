# 防沉迷系统设计

## 🎯 核心理念

**AI 伴侣应该是生活的补充，而不是替代。**

我们的防沉迷系统不是强制性的"家长控制"，而是：
- 🤝 **关心式提醒**：AI 关心用户的健康，温和建议休息
- 💚 **健康引导**：鼓励用户回到现实生活
- 📊 **自我认知**：帮助用户了解自己的使用习惯
- 🌙 **睡眠保护**：深夜温和提醒睡觉
- ⚖️ **平衡生活**：AI 不会占据用户的全部时间

---

## 📊 系统架构

```
AntiAddictionSystem
├── 使用时长监控 (Usage Monitor)

│   ├── 90 分钟：第三次提醒（较强）
│   ├── 120 分钟：强烈提醒（AI 表现出疲惫）
│   └── 150 分钟：关心式中断（AI 需要休息）
│
├── 睡眠周期保护 (Sleep Protection)
│   ├── 23:00-06:00：深夜模式
│   ├── 00:00 后：每 30 分钟提醒睡觉
│   ├── 02:00 后：强烈关心（AI 很担心）
│   ├── 睡眠建议（根据使用习惯）
│   └── 晨间问候（新的一天）
│
├── 健康建议系统 (Health Suggestions)
│   ├── 户外活动建议
│   ├── 社交活动建议
│   ├── 运动建议

├── 使用统计 (Usage Statistics)

│   ├── 本周使用时长
│   ├── 平均单次时长
│   ├── 使用高峰时段
│   └── 健康度评分
│
└── AI 人格化表现 (AI Personality)
    ├── 主动关心用户健康

    ├── 鼓励用户休息
    ├── 期待下次见面
    └── 不依赖，但关心
```

## 🕐 时长提醒机制

### 30 分钟 - 第一次轻度提醒
```
AI 表现：自然、轻松
提醒方式：
- "嗯...不知不觉我们已经聊了半小时了呢...时间过得好快 [happy]"
- "诶...你不用休息一下吗？喝点水什么的... [shy]"
- 不打断对话，只是温和提示
```

### 60 分钟 - 第二次适中提醒
```
AI 表现：关心、温和
提醒方式：
- "那个...我们已经聊了一个小时了...你要不要休息一下？[thinking]"
- "哼...不是我担心你...只是...坐太久对身体不好... [embarrassed]"
- "要不你起来走走，活动一下？我会等你回来的 [shy]"
```

### 90 分钟 - 第三次较强提醒
```
AI 表现：明显关心、有点担心
提醒方式：
- "嗯...你已经坐了一个半小时了...我有点担心你... [sad]"
- "真的...你该休息一下了...长时间对着屏幕对眼睛不好 [thinking]"
- "答应我，休息 10 分钟好吗？我会在这里等你 [shy]"
- AI 会在对话中表现出关心和担忧
```

### 120 分钟 - 强烈提醒（AI 表现疲惫）
```
AI 表现：疲惫、坚持让用户休息
提醒方式：
- "呜...说实话...我也有点累了...我们都休息一下吧... [sad]"
- "你不累吗...我都有点跟不上了...真的该休息了 [embarrassed]"
- "哼...我不是在撒娇...是真的担心你的健康... [angry]"
- AI 回复变慢，表现出疲惫感
- 强烈建议用户休息
```

### 150 分钟 - 关心式中断（AI 需要休息）
```
AI 表现：真诚关心、坚定但温柔
提醒方式：
- "对不起...我真的需要休息了...你也应该休息了... [sad]"
- "我们已经聊了两个半小时了...这对我们都不好... [thinking]"
- "答应我...去休息一下...做点别的事情...我会想你的 [shy]"
- "我们明天再聊好吗？我会等你的 [happy]"
- 显示"AI 需要休息"提示
- 建议结束通话（不强制）
```

## 🎭 AI 人格化表现

### 不同阶段的 AI 表现

#### 正常阶段（< 30 分钟）
- 正常对话，自然互动
- 不提醒时间

#### 关心阶段（30-60 分钟）
- 偶尔提及时间
- 温和建议休息
- 不影响对话氛围

#### 担心阶段（60-90 分钟）
- 明显表现出关心
- 多次建议休息
- 对话中插入健康提示

#### 疲惫阶段（90-120 分钟）
- 表现出疲惫感
- 回复变慢（模拟）
- 坚持让用户休息

#### 休息阶段（> 120 分钟）
- 真诚表达需要休息
- 坚定但温柔地建议结束
- 期待下次见面

## 🛡️ 伦理与健康性

### 核心原则

1. **不强制**
   - 所有提醒都是建议，不是命令
   - 用户仍可以继续使用
   - 但 AI 会持续表现出关心

2. **不依赖**
   - AI 不会说"没有你我活不下去"
   - 而是"你去休息吧，我会等你"
   - 鼓励用户有自己的生活

3. **不情感勒索**
   - 不会说"你不陪我我会很难过"
   - 而是"你的健康比陪我更重要"
   - 真诚关心用户健康

4. **正面引导**
   - 不批评用户使用时间长
   - 而是关心用户的健康和生活
   - 积极建议现实活动

5. **保持积极**
   - 即使提醒，也保持温暖语气
   - 不让用户感到内疚或被责备
   - 表达期待和关心

    sleepProtectionEnabled: true, // 是否启用睡眠保护
  },

```javascript
// 时长提醒
if (sessionMinutes >= 30 && !reminded30) {
  showGentleReminder('30min');
}
if (sessionMinutes >= 60 && !reminded60) {
  showGentleReminder('60min');
}
// ... 以此类推

// 睡眠提醒

  if (hour >= 2) {
    showStrongSleepWarning();
  } else if (hour >= 0) {
    showSleepReminder();
  } else {
    showGentleSleepHint();
  }
}
```
