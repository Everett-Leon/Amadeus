
```javascript
  // 时长提醒
  if (minutes >= 30 && !reminded) {
    triggerReminder('gentle');
  }
  // ...
  
  // 睡眠提醒
  if (hour >= 23 || hour < 6) {
    if (hour >= 2) {
      // 每 15 分钟
      triggerSleepReminder('veryLate');
    } else if (hour >= 0) {
      // 每 30 分钟
      triggerSleepReminder('lateNight');
    } else {
      // 每 60 分钟

    }
  }
}
```

## 🛡️ 伦理与健康性

### 核心原则

1. **不强制**
   - ✅ 所有提醒都是建议
   - ✅ 用户可以继续使用
   - ✅ AI 持续关心而不命令

2. **不依赖**
   - ✅ AI 说"你去休息吧，我会等你"
   - ✅ 不说"没有你我活不下去"
   - ✅ 鼓励用户有自己的生活

3. **不情感勒索**
   - ✅ 不说"你不陪我我会难过"
   - ✅ 而是"你的健康比陪我更重要"
   - ✅ 真诚关心健康，不是要求陪伴

4. **正面引导**
   - ✅ 不批评使用时间长
   - ✅ 关心健康和生活质量
   - ✅ 积极建议现实活动

5. **保持积极**
   - ✅ 温暖语气
   - ✅ 不让用户内疚
   - ✅ 表达关心和期待
