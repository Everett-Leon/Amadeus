# 背景图片上传功能说明

## ✅ 已完成的修改

### 1. 添加了背景图片上传功能

**欢迎页新增**：
- 5 个背景图片上传槽位（背景 1-5）
- 每个槽位可以独立上传图片
- 上传后会显示预览
- 配置会自动保存到 localStorage

**通话中切换**：
- 点击背景按钮可以切换已上传的背景
- 如果没有上传图片，会显示默认渐变色（兜底）

### 2. 图层结构（正确）

```
┌─────────────────────────────────┐
│  #bg-layer (背景层)              │  ← 最底层
├─────────────────────────────────┤
│  #character (人物立绘)           │  ← 中间层
│    ├─ #char-base (基础立绘)     │
│    └─ #char-expr (表情层)       │
├─────────────────────────────────┤
│  特效层 (粒子、气泡等)           │  ← 最上层
└─────────────────────────────────┘
```

**CSS z-index**：
- 背景层：z-index: 1
- 人物立绘：z-index: 2
- 特效层：z-index: 3+

✅ **人物立绘会覆盖在背景之上**

## 📋 使用方法

### 上传背景图片

1. **打开欢迎页**
2. **找到"🖼️ 背景图片（可选）"区域**
3. **点击任意槽位**（背景 1-5）
4. **选择图片文件**（推荐 1920x1080 或更高）
5. **等待上传完成**（会显示预览）

### 切换背景

**通话中**：
1. 点击右下角的"背景"按钮
2. 选择已上传的背景
3. 背景会平滑切换

## 🎨 推荐图片规格

### 尺寸
- **推荐**：1920x1080 (Full HD)
- **最佳**：2560x1440 (2K) 或 3840x2160 (4K)
- **比例**：16:9（横向）

### 格式
- **推荐**：JPG（文件小，加载快）
- **可选**：PNG（支持透明背景，但文件大）

### 文件大小
- **建议**：< 5MB（加载速度快）
- **最大**：50MB（系统限制）

### 风格建议
- 柔和的光线（避免过亮或过暗）
- 清晰的焦点（避免过于模糊）
- 与人物立绘风格匹配（动漫风格 or 真实照片）

## 🔧 技术细节

### 数据存储

**localStorage 结构**：
```javascript
{
  backgroundImages: {
    'bg-1': '/uploads/1234567890-abc123.jpg',
    'bg-2': '/uploads/1234567891-def456.jpg',
    // ...
  },
  background: 'bg-1',  // 当前选中的背景
  // ...
}
```

### 文件上传

- 上传到服务器的 `uploads/` 目录
- 文件名格式：`时间戳-随机ID.扩展名`
- 返回 URL：`/uploads/文件名`

### 兜底机制

如果某个背景槽位没有上传图片，会显示默认渐变色：
- 背景 1：紫色渐变（原卧室）
- 背景 2：棕色渐变（原图书馆）
- 背景 3：绿色渐变（原公园）
- 背景 4：深蓝渐变（原夜空）
- 背景 5：粉色渐变（原樱花）
