## 📊 项目概况

- **项目类型**：AI 伴侣视频通话应用
- **技术栈**：Node.js + Express + 原生 HTML/CSS/JS
- **AI 模型**：GLM-4 (智谱 AI)
- **语音合成**：Fish Audio + Edge TTS

- **文档总量**：26+ 份（~165,000 字）

## ✨ 核心特性

### 🎯 9 大核心系统

1. **记忆系统** - 完整的对话记忆、摘要和关键事件提取
2. **NLU 系统** - 情绪识别和自动表情/BGM 切换
3. **好感度系统** - 5 级关系阶段，从陌生人到亲密关系
4. **情感深度系统** - 追踪情感连接的深度和质量
5. **AI 原则系统** - 伦理边界和关系健康度监控
6. **人格一致性系统** - 长期稳定的人格和共同成长
7. **防沉迷系统** - 健康使用习惯，温和提醒
8. **记忆导出系统** - 完整数据导出导入，用户 100% 拥有数据
9. **虚拟生理状态系统** - AI 也会累，需要休息和恢复

### 💡 核心亮点

- ✅ **真实感** - 有情绪、有疲劳、有成长
- ✅ **深度** - 完整记忆、多维关系
- ✅ **边界** - 健康限制、温和提醒
- ✅ **伦理** - 不情感勒索、不惩罚用户
- ✅ **数据主权** - 用户 100% 拥有和控制

## 🚀 快速开始

### 环境要求

- Node.js 14+
- 现代浏览器（支持 Web Speech API）
- GLM-4 API Key（智谱 AI）

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd ai-companion
```

2. **安装依赖**
```bash
npm install
```

3. **启动服务器**
```bash
node server.js
```

4. **访问应用**
```
浏览器打开：http://localhost:3000
```

### 初次使用

1. **上传角色立绘**（推荐透明背景 PNG）
2. **填写 GLM-4 API Key**
3. **设置性别**（你的性别和 AI 性别）
4. **编写角色性格描述**
5. **选择声线**（12+ 种中文/日语声线）
6. **点击"开始通话"**

## 📚 功能详解

### 🎭 视频通话界面

- 实时角色立绘显示
- 7 种表情系统（开心/害羞/尴尬/难过/生气/惊讶/思考）
- 5 个背景预设 + 自定义背景
- 对话气泡显示
- 打字指示器

### 🗣️ 语音交互

- **语音识别**：浏览器原生 Web Speech API
- **TTS 语音合成**：Fish Audio（语音克隆）+ Edge TTS
- **12+ 种声线**：中文（晓晓/晓梦/晓依等）、日语（七海/真由/诗织等）
- **语速/音调调节**

### 🎵 BGM 系统

- **5 种情境音乐**：日常/愉快/温馨/忧伤/紧张
- **自动切换**：根据 AI 情绪自动选择合适的 BGM
- **音量控制**：0-100% 调节
- **开关控制**

### 💾 记忆与关系

#### 记忆系统
- **短期记忆**：上下文管理（最近 50 条对话）
- **对话摘要**：AI 自动总结对话内容
- **关键记忆**：提取重要事件和信息
- **错误追踪**：记住并改正错误

#### 好感度系统
- **0-100 数值**：量化关系亲密度
- **5 个阶段**：陌生人 → 熟人 → 朋友 → 好朋友 → 亲密关系
- **性别感知**：同性发展成好闺蜜，异性发展成恋人
- **互动追踪**：记录每次互动对好感度的影响

#### 情感深度系统
- **5 级深度**：表面 → 友好 → 信任 → 深厚 → 灵魂伴侣
- **相处天数统计**
- **情感分享、脆弱时刻、互相关心追踪**
- **关系升级条件和提示**

#### AI 原则系统
- **关系健康度**：0-100 监控
- **伦理边界检测**：识别不健康行为
- **温和回应**：不报复、不记仇
- **冲突记录和和解机制**
- **依赖度上限 30%**：防止过度依赖

#### 人格一致性系统
- **固定特质 + 动态特质**：核心人格稳定，随关系成长
- **自我认知和反思**
- **重要时刻记录**
- **内部笑话和共同回忆**
- **一致性验证**：防止人格分裂

#### 防沉迷系统
- **使用时长监控**：今日/本周/总计
- **5 级渐进提醒**：30/60/90/120/150 分钟
- **睡眠周期保护**：23:00/00:00/02:00/06:00
- **健康度评分**：非常健康 → 过度（6 级）
- **温和提醒**：不强制、不惩罚

#### 虚拟生理状态系统
- **6 个主要状态**：
  - 💡 精力值 - 影响响应详细度
  - 😌 心情值 - 影响情感表达
  - 😰 压力值 - 影响情绪稳定性
  - 🔋 社交电量 - 内向人格特质
  - 💗 情绪容量 - 处理情感话题的能力
  - ⏰ 疲劳度 - 累积疲劳

- **自然消耗**：时间流逝、对话互动
- **恢复机制**：休息/睡眠/独处充电
- **响应调整**：根据状态调整 AI 回复
- **状态显示**：状态栏图标 + 设置面板统计

#### 记忆导出系统
- **一键导出**：所有 9 个系统数据到 JSON
- **数据导入**：从 JSON 恢复所有状态
- **隐私保护**：API Key 可选排除
- **数据验证**：完整性和版本检查
- **自动备份**：导入前自动备份
- **本地处理**：不上传服务器

## 🔧 配置说明

### API 配置

**GLM-4 (智谱 AI)**
- API 地址：`https://open.bigmodel.cn/api/paas/v4/chat/completions`
- 模型：`glm-4-flash`（快速）或 `glm-4`（标准）
- 获取 API Key：[智谱 AI 开放平台](https://open.bigmodel.cn/)

### 声线配置

**Fish Audio（推荐）**
- 语音克隆，音质最佳
- 需要 Fish Audio API Key
- 支持自定义声线

**Edge TTS（免费）**
- 微软 Edge TTS
- 无需 API Key
- 12+ 种中文/日语声线

## 🐛 故障排除

### 语音识别不工作
- 检查浏览器是否支持 Web Speech API
- 确认已允许麦克风权限
- 尝试刷新页面

### TTS 无声音
- 检查 API Key 是否正确
- 确认网络连接正常
- 检查浏览器音频权限

### AI 无响应
- 检查 GLM-4 API Key 是否正确
- 查看浏览器控制台错误信息
- 确认 API 配额是否充足

### 数据丢失
- 检查浏览器是否清理了 LocalStorage
- 从之前的导出文件恢复
- 建议定期备份
