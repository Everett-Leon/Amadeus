# 表情图片文件夹

## 文件结构

请将生成的 7 个表情图片按以下命名保存到此文件夹：

```
expressions/
├── happy.png       # 😊 开心
├── shy.png         # 😳 害羞
├── embarrassed.png # 😣 尴尬
├── sad.png         # 😢 难过
├── angry.png       # 😠 生气
├── surprised.png   # 😲 惊讶
└── thinking.png    # 🤔 思考
```

## 图片要求

- **格式**：PNG（支持透明背景）
- **背景**：透明
- **尺寸**：与原立绘一致（建议 512x768 或更高）
- **风格**：保持与原立绘一致的画风

## 使用方法

生成并保存图片后，在 AI Companion 应用的欢迎页：
1. 展开"🎭 角色表情（可选）"
2. 为每个表情上传对应的图片
3. 开始通话后，AI 会根据情绪自动切换表情

## 注意事项

- 确保所有图片背景透明
- 保持角色的一致性（发型、服装、配色等）
- 表情变化要明显但不夸张
- 文件名必须严格按照上述命名（小写，无空格）
